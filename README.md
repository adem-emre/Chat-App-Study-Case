I customized faker package for generate random hour (only digits) with **justTime()** method<br>
To use this method you should add this package to your **pubspec.yaml** file

```yaml
dependencies:
  flutter:
    sdk: flutter
    
  faker:
      git:
        url: https://github.com/adem-emre/faker.git
        ref: only-digits
```
